# Dossier Images

Ce dossier contient toutes les ressources graphiques de l'application.

## Structure recommandée :

```
images/
├── logos/           # Logos de l'application
├── icons/           # Icônes personnalisées
├── backgrounds/     # Images de fond
├── avatars/         # Photos de profil utilisateur
├── products/        # Images de produits
└── misc/           # Autres images diverses
```

## Formats recommandés :

- **PNG** : Pour les logos et icônes avec transparence
- **JPG/JPEG** : Pour les photos et images sans transparence
- **SVG** : Pour les icônes vectorielles (recommandé)
- **WebP** : Format moderne pour de meilleures performances

## Bonnes pratiques :

1. **Optimisation** : Compresser les images avant de les ajouter
2. **Nommage** : Utiliser des noms descriptifs (ex: `logo-header.svg`)
3. **Tailles** : Fournir plusieurs tailles si nécessaire
4. **Alt text** : Toujours ajouter des descriptions pour l'accessibilité

## Outils recommandés :

- [TinyPNG](https://tinypng.com/) - Compression d'images
- [SVGO](https://jakearchibald.github.io/svgomg/) - Optimisation SVG
- [Squoosh](https://squoosh.app/) - Conversion et compression
